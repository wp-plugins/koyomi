Display current moon phase, dates, and Japanese old dates with graphical images.

== Description ==

To show current moon phase, dates, and Japanese old dates. Also, this shows Japanese 'roku-you' old six-day-based calendar with kanji character (you do not install Japanese font, it is graphic). Pointing it shows description.


== Installation ==

Place 'koyomi' folder in your /wp-content/plugins/ directory and put every thing init.
Then activate this through the administration panel, and then go to the widget panel and
drag it to where you would like to have it!
